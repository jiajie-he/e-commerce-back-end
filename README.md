# e-commerce-back-end

## Description
Built the back end for an e-commerce site. Configured Express.js API to use Sequelize to interact with a MySQL database.

## Usage
View, create, update, and delete data in database

## Installation
### On your terminal:
- npm i
- mysql -u root -p
- (enter your password)
- SOURCE db/schema.sql
- quit
- npm run seed
- node server.js
